Vercel CLI로 환경 변수 설정하기:

1. 로그인:
   vercel login

2. 프로젝트 연결:
   vercel link

3. 환경 변수 추가:
   vercel env add NEXT_PUBLIC_SUPABASE_URL production
   값 입력: https://beaufzsstxkmtzikkban.supabase.co

   vercel env add NEXT_PUBLIC_SUPABASE_ANON_KEY production
   값 입력: sb_publishable_ykNCQz7CU-H_18I_bmTtVA_yIKYKSvN

4. 재배포:
   vercel --prod
